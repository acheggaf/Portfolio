Presentation

Dans une équipe de 4 étduiants nous avons développé un site web qui permet à une artiste de montrer ses ouvres et d'avoir la possibilité de les vendre. Nous avons utilisé le modèle MVC (modèle vue controlleur) pour construire le site. Pour ce qui concerne le front-end nous avons fait usage de HTML5, CSS et jQuery. Pour ce qui concerne le back-end nous avons utilisé des servlets java dans un serveur Apache. Ma mission principale était de travailé sur le front-end, m ais aprés avoir fini avec ça j'ai travaillé sur la création de plusieurs servlets.

Admin

Version mobile

Dans notre site nous avons introduit un ensmeble de droit d'entrée admin, client et guest (utilisateur sans compte). l'admin possède des droits qui lui permet de pouvoir publier et de gèrer avec liberté tous les aspects du site.

Slider

Le Slider a été constuit en mettant plusieurs images (récupérées dynamiquement depuis une BD) les une sur les autres et en utilisant CSS, seulement l’image qui fait partie de la classe "active" 2 aura une opacité de 1 (les autres 0), et avec un timer dans Javascript on défile l’option class "active" sur l’ensemble des images. On a également ajouter l’option de pouvoir changer d’image à partir d’un ensemble de cercle qui indique la position actuelle parmi le reste

Responsive

Version mobil

Nous avons pris le temps de rendre notre site responsive (grace a un melange de CSS et Javascript) ce qui veut dire que les smartphones auront une version qui va faciliter la navigation, nous avons fait cela en cachant la navbar dans un menu qui se défile lorsque l’utilisateur clique sur un button. Nous avons aussi adapté la taille du text en utilisant CSS pour conserver des dimensions correctes.
