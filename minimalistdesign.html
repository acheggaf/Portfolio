Minimalist design

13/02/2022

Presentation

L'UI design est une section du développement logiciel qui me plaît énormément puisque selon moi un logiciel avec une interface graphique attirante est aussi important qu'avoir un logiciel performant. Le design minimalist est probablement mon type préféré. Et grâce a la framewrok JUCE que j'utilise pour développer du logiciel je suis capable d'avoir une très grande libérté sur la conception de l'interface graphique de mes projets. Dans mon projet Vibrato Effect j'ai décidé d'utiliser un style minimalist et de représenter les paramètres avec des formes abstraites avec des effets spécials.

Dans ce type de design le choix de couleur est trés important car il ne faut pas avoir énormément de couleur, de préférence deux ou trois maximums.

Un autre projet dans lequelle j'utilise ce type de design est mon halftime effect, dans ce logiciel j'ai utilisé le contraste entre le blanc et le noir et puis en utilisant un modèle 3D d'un amplificateur j'ai rendu le design beaucoup plus intéressant. Ce type de UI est souvent vu chez les sites web d'Apple.
